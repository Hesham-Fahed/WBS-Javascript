PHP Brainstorming

Netzwerk
Rechnernetz
Internet (aka Neuland)

Client
  Nutzt / Konsumiert Dienste
  zB Browser, Handy-App, Toaster mit Internet, Alexa, ...
Server
  Stellt Dienste bereit
  Kann den Rechner oder einen Service bezeichnen

Protokoll - Legt die Art/Syntax/Vokabular der Kommunikation fest
  TCP - Alle Daten kommen richtig an, Port
  IP - Daten kommen am richtigen Rechner an, IP
  DNS - Domain Name Service - Konvertiert Domain Namen in IPs
  Port - Daten kommen am richtigen Service an
  HTTP / HTTPS  - Protokoll von Web-Client/-Server
    GET, POST - HTTP Verben
  weitere: (S)FTP, SSH, Telnet

HTTP Request(-Response) Cycle
  Request - Anfrage auf eine Ressource, zB http://bla.de/meine/pizza/12
  Response - Antwort mit Code und ggf Daten, zB 200 { status: success }, oder 404

HTML - Auszeichnungssprache, Struktur des Dokuments, DOM ~ Baum
CSS - Beschreibungssprache um Elemente des DOMs zu Stylen
JavaScript - Skriptsprache zur Manipulation von Daten und des DOMs
PHP - Skriptsprache zur Manipulation von Daten, der DB und zum Bauen von Antworten
Scriptsprache - werden für uns unsichtbar kompiliert, wird oft interpretiert

Frontend - Clientseitig, sieht der Nutzer
Backend - Serverseitig, sieht der Nutzer nicht
